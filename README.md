# Pytorch-GP-VAE

This is a pytorch implementation for the article [GP-VAE: Deep Probabilistic Time Series Imputation](https://arxiv.org/abs/1907.04155)
The implemenation of the models was done by [Pytorch Lightning](https://www.pytorchlightning.ai) and the configuration of the project was done by [Hydra package](https://hydra.cc)
